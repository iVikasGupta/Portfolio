# Personal Portfolio Website

This repository contains my **professional portfolio website**, developed using **React** and **Vite**.
The website presents my skills, projects, experience, and contact information through a clean, responsive, and modern user interface.

---

## Live Portfolio

**Website:** [https://vikas-gupta-portfolio-five.vercel.app/](https://vikas-gupta-portfolio-five.vercel.app/)

---

## Overview

This portfolio serves as a centralized platform to highlight:

- Professional background and work experience
- Technical skills and tools
- Selected projects with clear descriptions
- Contact and communication details

The application is designed with a strong focus on performance, scalability, maintainability, and user experience.

---

## Key Features

- **React** for a modular, component-based architecture
- **Vite** for fast development and optimized production builds
- Fully **responsive design** across desktop and mobile devices
- Clean, modern, and professional UI
- **Hot Module Replacement (HMR)** for efficient development workflow
- **ESLint** integration to maintain code quality and consistency
- Deployed on **Vercel** for reliable and fast hosting
